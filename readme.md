# Slider Navigation 🧭👈👉

Neste projeto criei um slider, podendo ser utilizado de diversas maneiras (fotos, cards, etc).
Feito em Javascript, com navegação de touch (mobile), por setas e por buttons navigation.
Todo responsivo, onde quando é feito o resize (alterar a largura da janela do navegador, as imagens se ajustam automaticamente), utilizando neste processo o debounce (evitar que um método seja invocado milhares de vezes).
Estilo básico somente para dar um talento na página, mas o principal foi aplicar Class Constructions e Heranças, js modules, etc.
